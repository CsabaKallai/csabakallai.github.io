A projektről:

A projekt .NET frameworkben készült, konzolos alkalmazás, egyszerű menüvel kiegészítve.

A projekt célja egy megfelelően rétegzett architektúra elkészítése volt, a felhasználói felülettől leválasztott üzleti logika és adatbáziskezelő dll-ekkel. A logika számára az adatbázis elérést repositoryn keresztül érem el a Repository tervezési mintának megfelelően. A project sql first megközelítést alkalmaz.

A teszteléshez a NUnit keretrendszert használtam, kiegészítve a Moq keretrendszerrel. Az üzleti logika metódusait teszteltem. A logika repository objektumát mockoltam.
